About this project

TheraVox provides real‑time AI emotion analysis from faces, text, and audio. It reduces neutral bias in vision with tuned thresholds and margins, uses a lightweight rule‑based model for text (negation, intensifiers, emoticons), and a fast heuristic audio pipeline. The web app offers a modern, responsive UI with live camera/mic capture and a Save Screenshot feature.
